# CSU Auto Connect

针对中南大学的校园网自动登录工具，防止回家断网后需要手动认证登录。
支持实验室/电信/联通/移动/校园网。  (联通、移动未测试)

## 下载与安装
1. 从 Release 下载 `csu_auto_connect-vX.Y.Z-windows-x64.zip`
2. 解压后双击 `csu_auto_connect.exe` 运行


## 使用方法
1. 打开软件后填账号/密码
2. 选择网络类型
3. 点 `保存配置`
4. 点 `开始`

窗口会缩到托盘，托盘右键可“打开设置 / 立即登录 / 退出”。
